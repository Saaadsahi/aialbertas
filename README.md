# AIAlberta

AIAlberta is a public-facing platform for practical AI delivery in Alberta.

It brings together:
- a marketing site for services and projects
- a community space for discussion and transmissions
- contact and inquiry flows
- admin tooling for platform management

## Overview

The project is built as a modern full-stack web app with:
- Next.js
- Supabase
- Tailwind CSS
- Stripe

## What It Includes

- Homepage, services, projects, about, community, and contact pages
- Authentication for users and admins
- Community forum posting, comments, and reactions
- Cinematic transmission experiences
- Admin management for users, payments, moderation, and coffee chats
